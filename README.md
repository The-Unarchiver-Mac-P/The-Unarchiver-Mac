## 🔍 Overview

<a href="#theUnarchiver">The Unarchiver</a> serves virtually every Mac user who occasionally or frequently encounters archive files beyond the basic ZIP format that macOS handles natively — anyone downloading software, receiving files from Windows users, or working with internationally-sourced archived content.

The <a href="#theUnarchiver">cross-platform file sharing use case</a> is extremely common — files shared from Windows users, software downloaded from non-Mac-focused websites, and archives created with Windows-standard tools frequently use RAR, 7-Zip, or other formats that macOS's native Archive Utility simply cannot open. The Unarchiver eliminates the friction of encountering an unsupported archive format and needing to find a solution. The <a href="#theUnarchiver">international content and filename encoding use case</a> particularly benefits users who download content from international sources — software, media, or documents from Japanese, Chinese, Russian, or other non-English-language sources where filename encoding issues would otherwise corrupt filenames upon extraction.

The <a href="#theUnarchiver">software download and installation use case</a> applies to the common pattern of downloaded software arriving as a compressed archive in a format requiring extraction before installation — The Unarchiver handles this routine task across the full range of formats software distributors might use. <a href="#theUnarchiver">General Mac utility use</a> applies to virtually any Mac user as a reliable, free, always-available tool for the recurring task of extracting compressed files regardless of source or format.</p>
